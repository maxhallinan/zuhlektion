# zuhlektion

Simple selection sort.


## Install

```
$ npm install --save zuhlektion
```


## Usage

```js
const zuhlektion = require('zuhlektion');

zuhlektion([ 9, 7, 4, 2, 1, 3, 8, 6, 5, ]);
//=> [ 1, 2, 3, 4, 5, 6, 7, 8, 9, ]
```


## API

### zuhlektion(compare, arr)

#### compare

Type: `Function`

#### arr

Type: `Array`

The array to sort.
